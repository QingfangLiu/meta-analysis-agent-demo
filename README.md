# Meta-Analysis Agent Demo

This repository hosts a public static demo of an AI-assisted systematic review and meta-analysis agent.

The demo shows one end-to-end agent run, including search, screening, outcome planning, extraction, risk-of-bias handling, synthesis, and benchmark evaluation views.

Live demo: https://qingfangliu.github.io/meta-analysis-agent-demo/

This repository is a published static mirror. Source code and pipeline development live in the private/source project repository.
